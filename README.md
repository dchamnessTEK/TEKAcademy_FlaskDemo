# TEKAcademy Flask Demo App
To start the app:  
* Set environment variables
** FLASKAPP
** FLASKENV

run "flask run" in terminal

Dependencies: \
“pip install flask” \
“pip install flask_sqlalchemy” \
“pip install flask_migrate” \
“pip install flask _wtf” \
“pip install wtforms" \
"pip install python-dotenv"
